比特就业课简介

## 1. 企业简介

比特教育科技有限公司，成立于2013年，现有员工 180 余人。共设12大核心岗位，公司成员大多来自百度，阿里，腾讯等顶级互联网公司。

是一家以技术推动 IT 学子就业的教育科技公司，目前推出三大课程体系，累计在读学员6万+人。

三大课程体系：比特就业课（C++研发方向、Java研发方向、测试开发方向）、比特算法竞赛课、比特鹏哥C语言训练营

## 2. 我们为谁服务？

- 本科在读学生、研究生在读学生

## 3. 我们和别人有何不同？

- 比特专注做大学生校招10年，且比特只做大学生校招，比特的各项课程和服务都是为了让大学生更好的学习、实习、就业，所以我们的课是更适合在校大学生的。

- 不同于别人的是其他机构会做社招，更适合已经毕业的人群
